## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm

### Installation

```bash
npm install
```

### Development

Start the local dev server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.

### Build

To build for production:

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Lint

```bash
npm run lint
```
